# Creating Agents

In Alyzai, agents are the core components that enable automation. Create agents to execute tasks based on specific triggers and workflows.

## Steps to Create an Agent
1. Access the **Agents** section in the dashboard.
2. Click on the **New Agent** button.
3. Fill in the required details:
   - Name: Choose a descriptive name for your agent.
   - Description: Provide a brief overview of its purpose.
   - Triggers: Select the events that will activate this agent.
4. Save your agent and test it to ensure it functions as expected.
5. Monitor the agent's performance through the dashboard.

## Best Practices
- Use clear and consistent naming conventions.
- Regularly update agents based on changes in workflows or integrations to enhance functionality.
